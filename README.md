## Introdução
Este relatório apresenta o processo de implementação de ferramentas na empresa Pharma, realizado por Beatriz. O objetivo do projeto foi elencar 3 serviços AWS, com a finalidade de realizar diminuição de custos imediatos.

## Descrição do Projeto
O projeto de implementação de ferramentas foi dividido em 3 etapas, cada uma com seus objetivos específicos. A seguir, serão descritas as etapas do projeto:

Etapa 1: 
- Amazon EC2 (Elastic Compute Cloud)
- Hospedagem e gerenciamento de máquinas virtuais na nuvem.
- A empresa atualmente utiliza servidores físicos para executar sistemas internos de controle de estoque, gestão de medicamentos e aplicações administrativas.
Com a migração para o Amazon EC2, os servidores físicos poderão ser substituídos por instâncias virtuais sob demanda.
Vantagens:

Eliminação de custos com aquisição de servidores.
Redução de despesas com energia elétrica e refrigeração.
Escalabilidade conforme a necessidade do negócio.
Alta disponibilidade dos sistemas.

Etapa 2: 
- Amazon RDS (Relational Database Service)
- Gerenciamento de banco de dados relacional.
- A empresa possui bases de dados contendo informações de medicamentos, pesquisas clínicas, fornecedores e processos produtivos.
O Amazon RDS permitirá a administração automatizada desses bancos de dados, incluindo backups, monitoramento e atualizações de segurança.
Vantagens:

Backups automáticos.
Alta disponibilidade.
Atualizações automáticas.
Maior segurança dos dados.

Etapa 3: 
- Amazon S3 (Simple Storage Service)
- Armazenamento seguro de arquivos e documentos.
- A empresa farmacêutica armazena documentos regulatórios, resultados de pesquisas, relatórios laboratoriais, imagens e backups corporativos.
O Amazon S3 será utilizado como repositório centralizado para esses arquivos.
Vantagens:

Armazenamento altamente durável.
Escalabilidade praticamente ilimitada.
Controle de acesso e criptografia.
Integração com outros serviços AWS.


## Conclusão
A implementação de ferramentas na empresa Pharma tem como esperado obter redução de investimentos em hardware, diminuição dos custos de manutenção, maior eficiência operacional e melhor aproveitamento dos recursos tecnológicos, o que aumentará a eficiência e a produtividade da empresa. Recomenda-se a continuidade da utilização das ferramentas implementadas e a busca por novas tecnologias que possam melhorar ainda mais os processos da empresa.
